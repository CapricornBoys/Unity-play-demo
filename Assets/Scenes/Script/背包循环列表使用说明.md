# Unity 背包循环列表 - 使用说明

## 📁 已创建的脚本

| 脚本 | 功能 |
|------|------|
| `LoopList.cs` | 核心循环列表组件（对象池复用，高性能） |
| `InventoryItemData.cs` | 背包物品数据结构 |
| `InventoryItemUI.cs` | 物品 UI 显示组件 |
| `InventoryDemo.cs` | 演示脚本（模拟 200 个物品数据） |

---

## 🛠️ 在 Unity 场景中的设置步骤

### 第一步：创建 ScrollView
1. 在 Hierarchy 右键 → `UI → ScrollView`
2. 选中 ScrollView，在 Inspector 里添加 `LoopList` 组件
3. 如果使用垂直列表，确保 `ScrollRect` 的 `Vertical` ✅、`Horizontal` ❌
4. 如果使用水平列表，反之

### 第二步：设置 Content
1. 选中 `ScrollView/Viewport/Content`
2. 添加 `Vertical Layout Group`（垂直）或 `Horizontal Layout Group`（水平）
3. 取消勾选 `Child Force Expand`（宽/高都取消）
4. `Child Alignment` 设为 `Upper Center`

### 第三步：制作物品模板（ItemTemplate）
1. 在 `Content` 下右键 → `UI → Image`，命名为 `ItemTemplate`
2. 调整大小（例如 400×90）
3. 在 `ItemTemplate` 下添加：
   - `Image`（图标，命名 `Icon`）→ 锚点左上，大小 80×80
   - `Text`（名称，命名 `NameText`）→ 锚点左中
   - `Text`（数量，命名 `CountText`）→ 锚点右下
4. 把 `ItemTemplate` 拖成 **预制体**（或直接留在 Content 下也行）
5. **取消勾选 ItemTemplate 的 `SetActive`**（模板默认隐藏）

### 第四步：绑定 LoopList 参数
在 `LoopList` 组件的 Inspector 里：
- `Item Template` → 拖入 `ItemTemplate`
- `Content` → 拖入 `Content`
- `Item Size` → 设为物品的宽/高度（如 100）
- `Direction` → 选 `Vertical` 或 `Horizontal`

### 第五步：挂载演示脚本
1. 把 `InventoryDemo.cs` 挂载到 `ScrollView` 上
2. 把 `ItemTemplate` 拖入 `Item Template` 字段
3. （可选）修改 `Item Count` 调整测试物品数量

### 第六步：运行
点击 Play，背包列表应该能正常滚动，且只实例化屏幕可见的物品（高性能）。

---

## 🎯 核心原理

- **对象池复用**：只实例化屏幕可见数量的物品（如 10 个），滚动时回收离开屏幕的物品并重新绑定新数据
- **Content 动态撑大**：根据数据总数动态设置 Content 的 Height/Width
- **滚动监听**：在 Update 里检测 anchoredPosition 变化，触发刷新

---

## 📝 注意事项

- `itemTemplate` 必须默认 `SetActive(false)`（脚本里会自动处理）
- 真实项目中，把 `InventoryDemo.cs` 换成你自己的数据来源（如从服务器/存档读取）
- 图标（`Sprite`）目前为 null，可后续绑定真实图标资源
- 水平列表暂未完全测试，垂直列表为主力使用场景
